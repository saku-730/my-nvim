# This repository is Saku's nvim setting.

## Directory structure

```
├── init.lua
├── lazy-lock.json
├── lua
│   ├── keymaps.lua
│   ├── options.lua
│   └── plugins
│       ├── hichunk.lua
│       ￤
└── README.md
```

- init.lua:import other lua files
- keymaps.lua:setting keymaps
- optinos.lua:some settings other than keymaps and plugins
- plugins dir:1 lua file for 1 plugin
